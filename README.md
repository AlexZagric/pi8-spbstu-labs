# СПбПУ - Программная инженерия - Группа №8
# Структура репозитория
На момент 13-10-2024 репозиторий имеет и должен сохранять следующую структуру:
```
.
└─ название_предмета
   ├─ теория_с_лекций
   │  └─ ...
   ├─ лабораторная_#
   │  ├─ теория
   │  │  └─ ...
   │  └─ фамилия_имя
   │     └─ ...
   └─ практика_#
      ├─ теория
      │  └─ ...
      └─ фамилия_имя
         └─ ...
```
